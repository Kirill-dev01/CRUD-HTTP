# Домашнее задание: CRUD-приложение «Заметки»

React-приложение для создания и управления заметками. Проект демонстрирует работу с асинхронными HTTP-запросами (REST API) и синхронизацию состояния клиента с сервером.

## Реализованный функционал

В приложении реализованы основные операции CRUD (Create, Read, Delete) без перезагрузки страницы:
* **Чтение (GET):** Загрузка списка заметок с сервера при монтировании компонента и по кнопке «Обновить».
* **Создание (POST):** Добавление новой заметки через форму с автоматическим обновлением списка. Предусмотрена валидация на пустой ввод.
* **Удаление (DELETE):** Удаление заметки по клику на крестик с отправкой запроса на сервер и обновлением UI.
* **Обработка состояний:** Использование хуков `useState` и `useEffect` для управления данными и жизненным циклом.
